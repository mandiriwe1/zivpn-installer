# 🚀 ZIVPN Installer

Auto installer + panel management untuk ZIVPN UDP VPN server.

---

## ⚡ Fitur

- Auto install ZIVPN server
- Auto generate config
- Auto SSL self-signed
- Systemd service ready
- Admin panel CLI (menu)
- Add / delete user
- List akun
- Status server (IP, ISP, CPU, RAM)
- Auto fix config jika rusak
- Cache IP & ISP (lebih cepat)

---

## 📦 Supported OS

- Ubuntu 20.04 / 22.04 / 24.04
- Debian 10 / 11 / 12
